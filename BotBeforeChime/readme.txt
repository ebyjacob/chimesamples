Summary:
This is an example of how you can connect a bot on your system through the directline api to a chime dispatcher.

Requirements: 
Basic knowledge of c#
Bot Framework Emulator(V4) to test a but running on a local machine
DirectLine api credentials of a Chime dispatcher including the Microsoft App ID, app Name, directLine secret generated through azure portal.  

Alternatives:
If you were developing this for a node.js implementation check out https://github.com/microsoft/BotFramework-DirectLineJS for DirectLine JS examples.
